# 🎟 השוברים שלי — PWA

## כיצד להתקין כאפליקציה על הטלפון

### שלב 1 — העלאה ל-GitHub Pages (חינמי)

1. כנס ל-**https://github.com** וצור חשבון (אם אין לך)
2. לחץ **"New repository"** (ירוק, למעלה מימין)
3. שם הריפו: `vouchers-app`  
   סמן ✅ **"Public"**  
   לחץ **"Create repository"**

4. בתוך הריפו לחץ **"uploading an existing file"**
5. גרור את כל 4 הקבצים האלו:
   - `index.html`
   - `manifest.json`
   - `sw.js`
   - `icon.svg`
6. לחץ **"Commit changes"**

7. עבור לטאב **Settings** → **Pages**
8. תחת "Source" בחר **"Deploy from a branch"**
9. Branch: **main** → **/ (root)** → לחץ **Save**

10. אחרי ~1 דקה, האפליקציה תהיה זמינה בכתובת:  
    `https://YOUR-USERNAME.github.io/vouchers-app/`

---

### שלב 2 — התקנה על הטלפון

**אנדרואיד (Chrome):**
1. פתח את הכתובת בכרום
2. תופיע הודעת "הוסף למסך הבית" — לחץ עליה
3. או: תפריט ⋮ → "הוסף למסך הבית"

**iPhone (Safari):**
1. פתח את הכתובת ב-Safari
2. לחץ כפתור שיתוף 📤
3. "הוסף למסך הבית" → "הוסף"

---

האפליקציה תעבוד **גם ללא אינטרנט** לאחר ההתקנה.
